# MdEditor 帮助

## 基本操作
- `Ctrl+N`：新建文件
- `Ctrl+O`：打开文件
- `Ctrl+Shift+O`：打开文件夹工作区
- `Ctrl+S`：保存
- `Ctrl+Shift+S`：另存为

## 模式切换
- `Ctrl+E`：切换编辑/阅读模式
- 编辑模式支持：源码模式 / 所见渲染

## 大纲与侧栏
- `Ctrl+Shift+T`：显示或隐藏左侧面板
- 左侧面板支持：文件列表 / 大纲 / 插件管理

## 插件
- 插件目录：`<安装目录>/plugins`
- 结构兼容：`manifest.json` + `main.js`，可选 `styles.css`
- 复制插件文件夹到目录后，点击“刷新插件列表”即可加载

## 导出
- 工具栏点击“导出 PDF”

## 说明
- 本软件专注 Markdown 文件编辑与阅读，不包含数据库或关系图谱功能。
